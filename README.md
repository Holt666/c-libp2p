# c-libp2p
Implementation of libp2p in C.

## build

* clone from github

```
> git clone https://github.com/Holt666/c-libp2p
> git submodule init
> git submodule update
```
or

```
> git clone --recursive https://github.com/Holt666/c-libp2p
```

* use cmake build

```
> mkdir build & cd build
> cmake ..
> make
```

* run demo

```
> ./c-libp2p-test
> 
```
